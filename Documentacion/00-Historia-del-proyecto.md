#Adaptacion Darkos para consola H50Pro

##Descipcion del proyecto

Proyecto de investigacion y adaptacion de Darkos para una consola portatil H50 pro

El objetivo es conseguir una instalacion estable y funcional, documentando todo el proceso de configuracion, modificaciones y pruebas realizadas

##Estado inicial

La consola presento problemas de compatibilidad con las configuraciones de Darkos.

Se realizaron pruebas con diferentes configuraciones hasta encontrar una solucion basada en la modificacion del archivo DTB.

##Avances realizado 
-Instalacion de Darkos
-Pruebas con diferentes modelos de hardware
-Modificaciones del archivo DTB.
-Creacion de un DTB funcional denominado darkos_pruebas
-Adaptacion de temas para la resolucion de la H50
