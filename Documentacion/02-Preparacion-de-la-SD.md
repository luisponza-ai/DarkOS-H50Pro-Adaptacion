# Preparación de la tarjeta microSD

## Objetivo

Preparar una tarjeta microSD para realizar una instalación completamente limpia de dArkOSRE en una consola H50 Pro.

Esta etapa busca asegurar que la tarjeta se encuentre en condiciones óptimas antes de escribir la imagen del sistema operativo, evitando problemas derivados de particiones, formatos o información residual de instalaciones anteriores.

## Sistema operativo utilizado

**Windows**

## Herramientas utilizadas

| Herramienta       | Versión | Función                                       |
| ----------------- | ------- | --------------------------------------------- |
| SD Card Formatter | 5.0.3   | Formateo y preparación de la tarjeta microSD. |

## Procedimiento

1. Se verificó la capacidad de la tarjeta microSD que sería utilizada para la instalación.
2. Antes de realizar cualquier modificación, se generó un respaldo de la información considerada importante para el proyecto.
3. Se ejecutó SD Card Formatter para eliminar la estructura existente de la tarjeta y preparar un medio completamente limpio.
4. Una vez concluido el proceso, se comprobó que la tarjeta estuviera lista para recibir una nueva imagen del sistema operativo.

## Resultado

La tarjeta microSD quedó preparada correctamente para iniciar una instalación limpia de dArkOSRE.

A partir de este punto, la preparación del medio de almacenamiento se considera finalizada y el siguiente paso consiste en grabar la imagen del sistema operativo utilizando Rufus.
